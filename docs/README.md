# ClaimMate Documentation

This folder contains project planning, integration notes, demo scripts, and milestone materials. The root `README.md` is the short project overview. `AGENTS.md` remains the source of truth for coding agents.

## Recommended Reading

1. `plan.md`: product direction, roadmap, and team scope.
2. `PROJECT_PROGRESS_AND_STRUCTURE.md`: current implementation status and repo structure.
3. `BACKEND_INTEGRATION_SUMMARY_AND_LOU_GUIDE.md`: backend API flow and frontend integration notes.
4. `YI_FRONTEND_API_EXAMPLE_ZH.md`: frontend API examples for policy, accident, report, and chat flows.
5. `AI_CORE_PLAN_ZH.md`: AI/RAG/dispute/deadline/chat plan.
6. `AUTH_AND_WEBSOCKET_KE.md`: auth, invites, memberships, and WebSocket notes.
7. `RUN_DEMO_ZH.md`: local demo and smoke-test guide.

Files ending in `_ZH.md` are Chinese team handoff documents.
